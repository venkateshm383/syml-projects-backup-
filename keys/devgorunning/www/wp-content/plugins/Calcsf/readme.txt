== Installation ==

1. Upload the `visdom-mortgage-calculator` directory (including all files within) to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

= Usage =

* Add the shortcode `[vmcalc]` to your page or post and configure default parameters.
