Easily create and manage  a piechart .Get a Fancy jQuery Pie Chart with a simple shortcode

== Description ==
This plugin will added to your wordPress side a nice and easy pie chart.you can change pie chart total area width,single pie chart area width,total area background colour,pie chart circle text colour and also change text colour  & other setting from "pie_chart" menu in dashboard. This plugin will enable pie chart your wordpress theme. You can embed pie chart via shortcode in everywhere you want, even in theme files .

Shortcodes

shortcode in dashboard page/post/widgets

[pie_chart_list category="category come to post*"]

shortcode in theme page

<?php echo do_shortcode('[pie_chart_list category="category come to post*"]'); ?>

*post category and attribute category are same


= Plugin Features = 

* you can change pie chart total area width
* you can change single pie width
* you can change background colour
* you can change pie chart circle text colour
* you can change pie chart text colour
* you can add category in your pie chart




== Installation ==

This section describes how to install the sr-piechirt-wp plugin and get it working.

It is nice and very easy to install,


1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'setting/sr-piechart-wp' menu in WordPress
3.After activating sr-piechart-wp , you can create your first chart by clicking "pie_chart" menu in Dashboard
4.After clicking the "pie_chart" menu in Dashboard you can create your first chart by add new post.In post title put any think witch you want.Ex(HTML5,CSS3,JQUERY,WOOCOMERCE,RESPONSIVE ETC)
5.In post content you should Wright "circles-1"
6.next post "post circles-2"
7.next post "post circles-3"
8.next post "post circles-4"
9.next post "post circles-5"
10.next post "post circles-6"
11.also you set the order attributes in the post
12.post limited is 6
13.you can set the category.when you set the category must fill the category attribute in short code
14.you can define your pie chart width,circle text colour,text colour,pie chart total area background colour to pie_chart > pie setting menu 
15. Use shortcode in page, post or in widgets any where in your theme


Shortcodes

shortcode in dashboard page/post/widgets

[pie_chart_list category="category come to post*"]

shortcode in theme page

<?php echo do_shortcode('[pie_chart_list category="category come to post*"]'); ?>

*post category and attribute category are same
